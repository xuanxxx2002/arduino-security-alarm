# Arduino RFID 防盜系統



## 專案簡介

本專案為一套基於 \*\*Arduino 的嵌入式防盜系統\*\*，整合 RFID 身份驗證與紅外線（PIR）入侵偵測，實現簡易門禁與警報機制。



系統採用 \*\*狀態機設計（State Machine）\*\*，確保在不同安全狀態下的穩定運作與可控行為。



TINKERCAD 



---



## 硬體元件



| 元件 | 功能 |
|------|------|
| Arduino UNO | 主控制器 |
| RC522 RFID 模組 | 身份識別 |
| PIR 人體感測器 | 移動偵測 |
| 蜂鳴器 | 警報輸出 |
| LED | 狀態顯示 |
| 電阻 / 麵包板 | 電路搭建 |



---



## 接線說明



### RFID RC522（SPI 通訊）



| RC522 | Arduino |
|------|--------|
| SDA  | D10 |
| SCK  | D13 |
| MOSI | D11 |
| MISO | D12 |
| RST  | D9  |
| VCC  | 3.3V |
| GND  | GND |



---



### PIR 感測器



| PIR | Arduino |
|----|--------|
| OUT | D2 |
| VCC | 5V |
| GND | GND |



---



### 輸出裝置



| 裝置 | 腳位 |

|------|------|
| 蜂鳴器 | D6 |
| LED | D7 |



---



### 狀態設計說明

| 狀態 | 說明 |
|------|------|
| DISARMED（解除） | 系統未啟動警戒，等待 RFID |
| ARMED（布防） | 系統監控環境，等待入侵 |
| ALARM（警報） | 偵測到異常，觸發警報 |
